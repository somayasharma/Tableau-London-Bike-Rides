# Tableau-London-Bike-Rides

Tableau Dashboard: London Bike Rides
Overview
This project analyzes London bike rides between 07/02/2016 and 25/05/2016 using Tableau. The dashboard provides insights into:
- The total number of rides within the given period.
- 50-day moving average trends.
- Correlation between temperature, wind speed, and ride frequency.
- Weather conditions affecting ridership.
- Peak hours for bike usage.

Features
- Interactive Dashboard: Users can explore data trends over time.
- Weather Impact Analysis: Examines the effect of clear vs. cloudy weather on ridership.
- Hourly Trends: Displays peak hours of bike rides.
- Moving Average Insights: Shows fluctuations in bike usage.

Files Included
- London_Bike_Rides.twbx: The Tableau Packaged Workbook (includes data & visualizations).
- Dashboard 1-4.pdf: A PDF version of the Tableau dashboard for quick reference.
- Screenshots: Contains images of the dashboard for preview.

How to Open
1. Install Tableau Public or Tableau Desktop.
2. Open `London_Bike_Rides.twbx` to interact with the visualization.
3. Use filters and interactive elements to explore insights.

Dataset
This project is based on public datasets of London bike-sharing statistics. If you want to recreate the dashboard:
- Use a **bike rental dataset** with attributes like `Date`, `Rides`, `Temperature`, and `Weather Conditions`.
